# Multi-Agent System Project

## Description
This project implements a modern Multi-Agent System (MAS) using
Multi-Agent Reinforcement Learning (MARL). Each agent combines
reasoning, searching, planning, and learning capabilities.

## Project Structure
- data/: raw dataset
- src/: source code of MAS
- results/: experimental outputs
- docs/: system design documentation

## Agent Capabilities
- Reasoning: state abstraction
- Searching: possible action exploration
- Planning: policy-based action selection
- Learning: reinforcement learning update

## How to Run
```bash
python src/train.py
python src/evaluate.py
